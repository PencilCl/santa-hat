# santa-hat
Add a Santa hat to a picture

## Environment
This project is based on OpenCV and PIL(Pillow)
so we need to install them
### example(ubuntu)
```bash
sudo apt-get install libcv-dev
sudo apt-get install python-opencv
sudo pip install Pillow
```
